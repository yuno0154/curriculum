# 📚 성취기준 추출 및 검색 시스템

2022 개정 교육과정 PDF 문서에서 성취기준을 자동으로 추출하고, 웹 대시보드를 통해 편리하게 검색할 수 있는 도구입니다.

---

## 🛠️ 프로그램 구성

### 1. 메인 실행 파일
- **`extractor.py`**: PDF 문서(`교육과정문서`, `achivement_files`)에서 성취기준 코드를 찾아 진술문을 추출합니다.
- **`run_server.bat`**: 웹 대시보드 서버를 실행합니다.

### 2. 데이터 파일 (자동 생성)
- **`data.json`**: 웹 대시보드에서 사용하는 데이터 파일입니다.
- **`성취기준_추출결과.xlsx`**: 추출된 모든 성취기준을 엑셀로 한눈에 확인할 수 있습니다.

### 3. 웹 인터페이스
- **`index.html`**, **`style.css`**, **`app.js`**: 성취기준 검색을 위한 웹 대시보드 구성 요소입니다.

---

## 🚀 사용 방법

### 1단계: 데이터 추출
1. 추출하고자 하는 PDF 파일을 `교육과정문서` 또는 `achivement_files` 폴더에 넣습니다. (이미 별책3, 4번 문서가 포함되어 있습니다.)
2. **`extractor.py`** 파일을 실행합니다.
   - 실행이 완료되면 `data.json`과 `성취기준_추출결과.xlsx` 파일이 생성됩니다.

### 2단계: 대시보드 확인
1. **`run_server.bat`** 파일을 실행합니다.
2. 브라우저 주소창에 `http://localhost:8000` 을 입력하여 접속합니다.
3. 과목 필터링 및 검색 기능을 통해 원하는 성취기준을 찾습니다.

---

## 📂 기타 안내
- **`archive/`** 폴더: 개발 과정에서 생성된 이전 버전 및 테스트용 스크립트들을 보관하는 장소입니다. 일반적인 사용 시에는 무시하셔도 됩니다.
- **`venv/`**: 파이썬 실행 환경(가상환경) 폴더입니다.
